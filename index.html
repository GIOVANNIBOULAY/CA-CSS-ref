BASIC SELECTORS

UNIVERSAL SELECTORS
DEFINITION Selects all elements. Optionally, it may be restricted to a specific namespace or to all namespaces.
SYNTAX * ns|* *|*
EXAMPLE * will match all the elements of the document.
type selectors
DEFINITION Selects all elements that have the given node name.
SYNTAX elementname
EXAMPLE input will match any <input> element
class selectors
DEFINITION Selects all elements that have the given class attribute.
SYNTAX .classname
EXAMPLE .index will match any element that has a class of "index".
id selector
DEFINITION Selects an element based on the value of its id attribute. There should be only one element with a given ID in a document.
SYNTAX #idname
EXAMPLE #toc will match the element that has the ID "toc".
attribute selector
DEFINITION Selects all elements that have the given attribute.
SYNTAX [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]
EXAMPLE [autoplay] will match all elements that have the autoplay attribute set (to any value).
